   URL: http://localhost:5000/api/users
   Metodo: GET
   Body: n/a

2. Utente ID 1
   URL: http://localhost:5000/api/users/1
   Metodo: GET
   Body: n/a

3. Tutte le meditazioni
   URL: http://localhost:5000/api/meditations
   Metodo: GET
   Body: n/a

4. Intervalli utente 1
   URL: http://localhost:5000/api/users/1/intervals
   Metodo: GET
   Body: n/a

5. Intervalli filtrati (per date e goal)
   URL: http://localhost:5000/api/intervals/filter?startDate=2026-03-01&endDate=2026-03-31&goalId=1
   Metodo: GET
   Body: n/a

6. Intervallo ID 1
   URL: http://localhost:5000/api/intervals/1
   Metodo: GET
   Body: n/a

--- POST (creare dati) ---

1. Nuovo utente
   URL: http://localhost:5000/api/users
   Metodo: POST
   Body (JSON):
   {
       "email": "giulia@email.com",
       "nome": "Giulia",
       "cognome": "Bianchi"
   }

2. Nuova meditazione
   URL: http://localhost:5000/api/meditations
   Metodo: POST
   Body (JSON):
   {
       "title": "Meditazione pomeridiana",
       "duration": 12
   }

3. Nuovo intervallo utente 1
   URL: http://localhost:5000/api/intervals
   Metodo: POST
   Body (JSON):
   {
       "startDate": "2026-04-01",
       "endDate": "2026-04-07",
       "userId": 1
   }

4. Associa meditazione ID 2 a intervallo ID 1
   URL: http://localhost:5000/api/intervals/1/goals
   Metodo: POST
   Body (JSON):
   {
       "goalId": 2
   }

5. Associa meditazione ID 3 a intervallo ID 1
   URL: http://localhost:5000/api/intervals/1/goals
   Metodo: POST
   Body (JSON):
   {
       "goalId": 3
   }

---
